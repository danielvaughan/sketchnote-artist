Details (Small Text):
- **Local, Private & Free AI Inference:**
    - Run LLMs on your own hardware (GPU/NPU).
    - Data stays on device = ultimate privacy.
    - No cloud dependencies, no API costs.
- **Easy Setup, Diverse Models, Cross-Platform:**
    - Simple CLI installation (`winget install`).
    - Supports Phi-3, Mistral, DeepSeek, etc.
    - Works on Windows, macOS, Linux (WSL2).
- **OpenAI Compatible: Code Local, Run Global:**
    - Drop-in replacement for OpenAI API.
    - Change `base_url` to local endpoint.
    - Use existing OpenAI SDK/LangChain code.
- **Real-World AI: From Chat to RAG Apps:**
    - CLI chat demonstrations.
    - Refactor RAG for fully offline PDF Q&A.
    - Streamlit web apps running locally.
